Configure servers and validators. Custom values are saved in your browser.

Terminology Server
FHIR $expand endpoint used to resolve ValueSet bindings (choice items, answer options).
Used by Answer Type modal when expanding ValueSets from a canonical URL.
FHIR Base Server
Base FHIR endpoint for resource lookups — used to search for Patients, Practitioners, and other referenced resources in the preview. Also used for SDC server operations (e.g. $populate via Answers ▾ → Fill from FHIR Server). Note: $populate requires a server implementing the SDC IG — standard public servers (HAPI, SmartHealthIT) do not support it. Try Matchbox for SDC operations.
Used for reference item type autocomplete. Leave blank to disable.
SDC Server
FHIR server implementing the SDC IG — used for $populate and other SDC server operations (Answers ▾ → Fill from FHIR Server). If blank, falls back to FHIR Base Server. Example: https://test.ahdis.ch/matchboxv3/fhir (Matchbox).
Used for $populate. Separate from FHIR Base Server which is for resource lookups.
CORS Proxy
Proxy worker that forwards requests to FHIR servers when CORS headers are not set. Required for browser-based validation against external servers.
Requests are forwarded as proxy?url=<encoded-target>. Leave blank to call servers directly.
NLM Clinical Tables API
NLM endpoint for searching clinical terminology (LOINC, SNOMED, ICD). No CORS proxy needed — NLM sends Access-Control-Allow-Origin: *.
Translation API
Endpoint used to auto-translate questionnaire text (Translate modal). Defaults to the unofficial Google Translate gtx endpoint (free, no key). Point this at a self-hosted gtx-compatible proxy if needed.
Query params (client=gtx&sl&tl&dt=t&q) are appended automatically. Leave blank to use the default Google endpoint.
Validators
Validation pipeline run on import and on demand. The built-in validator is always available. External validators POST the FHIR Questionnaire to a FHIR server's $validate endpoint.
Active Configuration Sources
Settings are resolved in priority order — higher sources override lower ones.
